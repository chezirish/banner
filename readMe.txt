Инструкция по использованию:
1. Загрузить на сервер файлы ajax.php, banner.php, banner_style.css и banner.js в однку папку.
2. В файле banner.js указать ссылки на banner_style.css и ajax.php c вашего сервера.
3. Подключить скрипт banner.js с сервера в тег head на сайта-партнера с атрибутом defer  ( <script defer src="http://my/server/banner.js"></script> ).
